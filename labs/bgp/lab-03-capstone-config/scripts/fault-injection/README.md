# Solution Reference -- BGP Lab 03 (Capstone I)

This capstone lab is clean-slate build-from-scratch. There are no fault
injection scenarios -- comprehensive troubleshooting is covered in Lab 04.

## Prerequisites

1. The lab is running in EVE-NG (`bgp/lab-03-capstone-config.unl`).
2. Initial configs (IP addressing only) have been pushed with
   `python3 ../../setup_lab.py --host <eve-ng-ip>`.

## Apply the Reference Solution

Use this once you have completed your own build to compare against the
reference end-state, or to skip ahead to Lab 04 with a known-good config.

```bash
python3 apply_solution.py
```
